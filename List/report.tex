\begin{lstlisting}[language=C++]  
List<int> lst1;  
List<int> lst2 {1,3,5,7,9};  
List<int> lst3 = lst2;  
List<int> lst4 = List<int>({2,4,6,8,10});  
const List<int> lst5(lst3);  
\end{lstlisting}  
  
通过以下代码测试 \texttt{push\_back} 和 \texttt{push\_front} 的调用：  
  
\begin{lstlisting}[language=C++]  
for (int i = 0; i < 10; ++i) {  
    if (i % 2 == 0)   
        lst1.push_back(i);  
    else   
        lst1.push_front(i);  
}  
lst1.push_back(13);  
\end{lstlisting}
